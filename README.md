# 🛡️ Secure Online Aptitude Exam System

> A secure, web-based aptitude examination platform developed using **Backend Development** and **Cyber Security** concepts. The system enables administrators to create and manage online exams while allowing students to take secure, timed assessments with instant result generation.

---

## 📌 Features

### 👨‍🎓 Student
- Secure Registration & Login
- JWT Authentication
- Timed Online Exams
- Multiple Choice Questions (MCQs)
- Instant Score Calculation
- View Exam History
- Responsive Dashboard
- Secure Logout

### 👨‍💼 Admin
- Admin Dashboard
- Manage Students
- Manage Questions
- Create & Schedule Exams
- View Results
- Generate Reports
- Monitor Active Exams

### 🔒 Security Features
- Password Hashing (BCrypt)
- JWT Authentication
- Role-Based Access Control (RBAC)
- SQL Injection Protection
- XSS Protection
- CSRF Protection
- Input Validation
- Rate Limiting
- Secure REST APIs
- Environment Variables (.env)

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/vdeekshith2007/Secure-Aptitude-Exam.git
```

### Navigate to Project

```bash
cd Secure-Aptitude-Exam
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=password
DB_NAME=examdb
JWT_SECRET=your_secret_key
```

### Run Application

```bash
npm start
```
